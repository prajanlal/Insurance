Smart Insurance Management System

Project Overview
This project is a simple web-based insurance management system that we built to make insurance services easier to use. The main idea was to create a platform where users can view their policies, raise claims, and manage everything in one place without confusion.

What the System Does
- Allows users to view and manage their insurance policies  
- Helps users submit and track claims  
- Provides a clean and easy-to-use dashboard  
- Includes a login system for secure access  
- We also planned a chatbot feature to assist users with basic queries 
 
Technologies Used

Frontend:
- HTML, CSS, JavaScript (for building the interface and interactions)

Backend:
- Python (planned for handling logic and APIs)

Database:
- MySQL (for storing user and policy data)


How the System Works?
When the user opens the application, they first see the landing page.  
From there, they can log in using the login page.  

After logging in, the user is taken to the dashboard, where they can:
- View policies  
- Check claims  
- Perform basic actions
